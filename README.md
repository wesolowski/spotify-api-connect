This is a Wrapper for [Spotify Web API PHP](https://github.com/jwilsson/spotify-web-api-php)


## Requirements
* PHP 7.1 or later.
* PHP [cURL extension](http://php.net/manual/en/book.curl.php) (Usually included with PHP).
